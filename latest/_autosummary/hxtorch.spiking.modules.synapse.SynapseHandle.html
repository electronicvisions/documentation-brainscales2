put(*tensors, **kwargs) → None

Fill the tensor handle with actual data given by tensors or kwargs. If tensors are given as positional arguments the tensors are assigned in the order given by class member ‘_carries’. Keyword arguments are assigned to the corresponding key. Therefore, the key has to be in ‘_carries.’

Parameters

  • tensors – Tensors which are assigned to the tensor handle. The given tensors are associated with the elements in ‘_carries’ in successive order.

  • kwargs (param) – Assigns the items in kwargs to the elements in the tensor handle associated with the corresponding kwargs keys.
