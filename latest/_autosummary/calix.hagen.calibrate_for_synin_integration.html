calix.hagen.calibrate_for_synin_integration

calix.hagen.calibrate_for_synin_integration(connection: pyhxcomm_vx.ConnectionHandle, target: Optional[calix.hagen.HagenSyninCalibTarget] = None, options: Optional[calix.hagen.HagenSyninCalibOptions] = None) → calix.hagen.HagenSyninCalibResult

Calibrate the chip for integration on synaptic input lines.

Calibrate CADC, synapse drivers, and synapse DAC bias.

Parameters

  • connection – Connection to the chip to calibrate.

  • target – Calib target parameters.

  • options – Further options for calibration.

Returns

Calib result for integration on the synaptic input lines.
