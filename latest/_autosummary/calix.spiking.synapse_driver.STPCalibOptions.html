calix.spiking.synapse_driver.STPCalibOptions

class calix.spiking.synapse_driver.STPCalibOptions

Bases: calix.common.base.CalibOptions

Set bias parameters for the STP circuitry.

All parameters can be configured per quadrant.

Parameters

  • i_ramp – Ramp current for STP pulse width modulation, in CapMem LSB.

  • v_stp – Voltage (STP state) where all drivers’ amplitudes are equalized at, in CapMem LSB. Should be chosen between v_charge and v_recover.
